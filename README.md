# 📖 About the Project

This project demonstrates the practical implementation of **Machine Learning algorithms** using the **Iris Dataset**.

The notebook walks through the complete Machine Learning pipeline—from importing the dataset and preprocessing the data to training, evaluating, and comparing multiple Machine Learning models using **Scikit-learn**.

It serves as a beginner-friendly yet practical introduction to real-world Machine Learning workflows.
